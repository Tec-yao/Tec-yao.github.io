mysql的常用命令

Posted on 2018-04-18

Mysql登录

-u用户名
-p密码
-P端口号
-h服务器名字

退出：exit quit \q

修改提示符

在登录时 使用–prompt+修改的提示符
在登录完后 使用prompt+修改提示符
二者选其一
修改提示符的意义在于更清晰的知道在操作的数据库是哪一个

MySQL语言规范

关键词与函数名称全部大写 数据库名称，表名称，字段名称必须全部小写
*SQL语句必须以分号为结尾

数据类型

整型

TINYINT 0到255 SMALLINT 0到65535
MEDIUMINT 0到16777215 INT 0到4294967295
*BIGINT 0到1844674073709551615

以上有符号对半折

浮点型

FLOAT[(M,D)] DOUBLE[(M,D)]

注：其中M代表数字总位数，D代表后面的小数点

字符型

CHAR(M) VARCHAR(M)
IINTTEXT TEXT
MEDIUMTEXT LONGIEXT
ENUM(‘value1’,’value2’,…) 枚举 SET(‘values1’,’values2’,…) 集合
M:表示可以指定大小

创建数据表

CREATE TABLE [IF NOT EXISTS] table_name(
clumn_name data_type,
…
)

注：clumn_name是列名，变量名。data_type 是变量类型
最后一个列名不用加逗号
